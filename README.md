# 🧠 KnowledgeMatch app
##### A cross platform mobile app written in Flutter

KnowledgeMatch is a mobile app that connects students who need help with 
academic topics to nearby experts who can support them — fast, friendly, 
and effective.

### 📲 Features:

- 🔍 Find nearby experts on university subjects
- 📅 Schedule meetings
- ✅ Verified student accounts
- 🌐 Built with Flutter & Firebase (cross-platform)
